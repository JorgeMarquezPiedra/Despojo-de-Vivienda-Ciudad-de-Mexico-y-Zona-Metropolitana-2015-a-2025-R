# Despojo-de-Vivienda-Ciudad-de-Mexico-y-Zona-Metropolitana-2015-a-2025-R
Análisis espacial del delito de despojo de viviendas en la Ciudad de México y Zona Metropolitana (2015-2025) con R. Utilizando datos oficiales del Sistema Nacional de Seguridad Pública (SNSP), se procesaron y filtraron los reportes de incidencia delictiva para la generación de cartografía.
